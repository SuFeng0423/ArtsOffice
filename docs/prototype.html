🔄 核心业务流程

请假审批流程

学生提交申请 → 班主任审核 → 教务审批（可选） → 通知家长（可选）

视频审核流程

学生上传视频 → pending 待审核 → approved / rejected → 教师提交点评

成绩科目维度

日常：练声·鉴赏·乐理·写作·理论 大考：视唱·声乐·器乐·总成·排名

⚙️ 技术栈建议
Vue 3 + Vite + Element Plus 微信原生 / Taro / uni-app FastAPI (Python) MySQL + Redis 腾讯云 COS / MinIO JWT + 微信 OAuth

开发优先级：P0 认证 & 人员管理 → P1 成绩 & 视频 → P2 请假 & 仪表盘 → P3 权限 UI & 通知推送
